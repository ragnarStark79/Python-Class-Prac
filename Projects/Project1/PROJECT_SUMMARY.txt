=====================================================
                  CUSTOMIZATION
=====================================================

CHANGE COLORS:
   Edit CSS variables in static/css/style.css
   Lines 2-15 contain all color definitions

UPDATE CONTENT:
   - Home: templates/home.html
   - Semester: templates/previous_semester.html
   - Portfolio: templates/portfolio.html

ADD PROJECTS:
   Copy a project-card div in portfolio.html
   Update category, title, description, and tags

CHANGE PORT:
   Edit main.py, line 16
   Change: port=5001 to your preferred port
